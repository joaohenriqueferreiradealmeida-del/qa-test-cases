# Test Cases – Login Feature

## Test Case 01 – Login with valid credentials
**Precondition:** User is registered  
**Steps:**
1. Access login page
2. Enter valid email
3. Enter valid password
4. Click on Login button

**Expected Result:** User should be redirected to dashboard

---

## Test Case 02 – Login with invalid password
**Precondition:** User is registered  
**Steps:**
1. Access login page
2. Enter valid email
3. Enter invalid password
4. Click on Login button

**Expected Result:** Error message should be displayed

---

## Test Case 03 – Login with empty fields
**Steps:**
1. Access login page
2. Leave email and password empty
3. Click on Login button

**Expected Result:** Validation message should be displayed
